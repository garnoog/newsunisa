คำจำกัดความพื้นฐานและประโยชน์ของเว็บไซต์

ความหมาย

    เว็บเพจ (อังกฤษ: web page, webpage)
     แปลเป็นไทยว่า หน้าเว็บ คือเอกสารเว็บชนิดหนึ่ง เหมาะสำหรับเว็ลด์ไวด์เว็บ และเว็บเบราว์เซอร์ เว็บเบราว์เซอร์จะแสดงเว็บเพจบนจอคอมพิวเตอร์หรืออุปกรณ์เคลื่อนที เว็บเพจก็คือสิ่งที่ปรากฏออกมา
แต่ศัพท์นี้ก็ยังหมายถึงแฟ้มคอมพิวเตอร์ที่มักจะเขียนเป็นเอชทีเอ็มแอล หรือภาษามาร์กลัปที่เทียบเคียงได้ ซึ่งมีลักษณะเด่นอันเป็นหลักก็คือ การจัดเตรียมข้อความหลายมิติ ที่จะนำไปสู่เว็บเพจอื่น ผ่านทางลิงก์
เว็บเบราว์เซอร์จะประสานงานกับทรัพยากรเว็บที่อยู่โดยรอบ เว็บเพจที่เขียน อาทิ สไตล์ชีต สคริปต์ และรูปภาพ เพื่อนำเสนอเว็บเพจนั้น

โฮมเพจ (อังกฤษ: home page)
     หมายถึง เว็บเพจหน้าแรกที่ปรากฏของแต่ละเว็บไซด์

เว็บไซต์ (อังกฤษ: website, web site หรือ site)
   änbsp; หมายถึง หน้าเว็บเพจหลายหน้า ซึ่งเชื่อมโยงกันผ่านทางไฮเปอร์ลิงก์ ส่วนใหญ่จัดทำขึ้น เพื่อนำเสนอข้อมูลผ่านคอมพิวเตอร์ โดยถูกจัดเก็บไว้ในเวิลด์ไวด์เว็บ หน้าแรกของเว็บไซต์ที่เก็บไว้ที่ชื่อหลักจะเรียกว่า โฮมเพจ เว็บไซด์

เว็บแอปพลิเคชัน (อังกฤษ : web application)
     คือโปรแกรมประยุกต์ที่เข้าถึงด้วยเบราว์เซอร์ (web browser ) ผ่านเครือข่ายคอมพิวเตอร์ อย่างอินเทอร์เน็ต หรืออินทราเน็ตโดยตรง ไม่ต้องโหลดแอปพลิเคชัน (Application) แบบเต็มๆ ลงเครื่องทําให้ใช้ทรัพยากรค่อนข้างต่ํา สามารถเปิดใช้งานได้ไว

    ปัจจุบันมี Web Application ที่หลากหลาย ทั้งในรูปแบบใช้งานระดับโลก ระดับมหาวิทยาลัย ไปจนถึงระดับบริษัท ก็มีเว็บแอปเป็นของตัวเอง ตัวอย่างเว็บแอปพลิเคชันได้แก่ เว็บเมลการพาณิชย์อิเล็กทรอนิกส์ การประมูลออนไลน์ กระดานสนทนา บล็อก

เว็บเบราว์เซอร์ (อังกฤษ : web browser)

     เบราว์เซอร์ หรือ โปรแกรมค้นดูเว็บ คือโปรแกรมคอมพิวเตอร์ ที่ผู้ใช้สามารถดูข้อมูล และโต้ตอบกับข้อมูลสารสนเทศที่จัดเก็บในหน้าเว็บที่สร้างด้วยภาษาเฉพาะ เช่น ภาษาเอชทีเอ็มแอล ที่จัดเก็บไว้ที่เว็บเซอร์วิซหรือเว็บเซิร์ฟเวอร์หรือระบบคลังข้อมูลอื่น ๆ
โดยโปรแกรมค้นดูเว็บเปรียบเสมือนเครื่องมือในการติดต่อกับเครือข่ายคอมพิวเตอร์ขนาดใหญ่ที่เรียกว่าเวิลด์ไวด์เว็บ

     เว็บเบราว์เซอร์ตัวแรกของโลกชื่อ เวิลด์ไรต์เว็บ ขณะเดียวกันเว็บเบราว์เซอร์ที่นิยมได้แก่เกิด โครม, มอซิลลา ไฟฟand, ซาฟารี และอินเทอร์เน็ตเอกซ์พลอเรอ

ไอพีแอดเดรส (อังกฤษ : IP address; หรือ อนเช่น ที ไท้, เลขไอพี)

    คือสัญลักษณ์เชิงหมายเลขที่กําหนดให้แก่อุปกรณ์แต่ละชนิด (เช่นคอมพิวเตอร์ เครื่องพิมพ์) ที่มีส่วนร่วมอยู่ในเครือข่ายคอมพิวเตอร์หนึ่ง ที่ใช้อินเทอร์เน็ตโพรโทคอลในการสื่อสารไอทีแอดเดรสทําหน้าที่สําคัญสองอย่าง
ได้แก่การระบุแม่ข่ายหรือส่วนต่อประสานเครือข่าย และการกําหนดที่อยู่ให้ตําแหน่งที่ตั้ง บทบาทของมันได้บรรยายไว้ว่า ใช้แสดงว่าเราค้นหาอะไรที่อยู่ใช้แสดงว่ามันอยู่ที่ไหน เส้นทางใช้แสดงว่าจะไปที่นั่นอย่างไรแต่เต็มผู้ออกแบบอินเทอร์เน็ตโพรโทคอล (br) ได้กําหนดไอพีแอดเดรสให้เป็นตัวเลข 32 บิต ซึ่งรู้จักในชื่อเลขที่อยู่ในรุ่น 4 (IPv6) และระบบนี้ยังคงมีการใช้งานอยู่ในปัจจุบัน อย่างไรก็ดีปัจจุบันอินเทอร์เน็ตเติบโตขึ้นอย่างมหาศาล และมีการคาดการณ์ว่าเลขที่อยู่ในบ้าน 4 จะถูกใช้หมดไป
เลขที่อยู่ไอทีรุ่นใหม่ จึงได้พัฒนาขึ้นใน ค.ศ. 1995 คือเลขที่อยู่ไอทีรุ่น 6 (IPv6) ซึ่งใช้ตัวเลข 128 บิดกําหนดที่อยู่และได้ทําให้เป็นมาตรฐานใน อาร์เอฟซี 2468 เมื่อ ค.ศ. 1998 ส่วนการนํามาใช้จริงนั้นเริ่มตั้งแต่กลางคริสต์ทศวรรษ 2000

ไอพีแอดเดรสเป็นเลขฐานสอง แต่ก็มักจะแสดงผลและเก็บบันทึกในไฟล์ข้อความที่มนุษย์สามารถอ่านได้ ตัวอย่างเช่น

ไอทีน 4 (IPv4) 172.16.254.1
ไอทีรุ่น 6 (IPv6) 2001:bd8:0:1234:0:567:8:1

อโดเมน (อังกฤษ : domain name)

หมายถึง ชื่อที่ใช้ระบุลงในคอมพิวเตอร์ (เช่น เป็นส่วนหนึ่งของที่อยู่เว็บไซต์ หรืออีเมลแอดเดรส)เพื่อไปค้นหาในระบบโดเมนเนมซีสเทม

ในทีรุ่น 4 (IP} โดเมนเนม
157.240.18.354/th facebook.com

เว็บเซิร์ฟเวอร์ (อังกฤษ : web server) สามารถมีได้ 2 ความหมาย คือ

    เครื่องคอมพิวเตอร์ซึ่งให้บริการที่เก็บเว็บไซต์ (Server) แล้วให้ผู้ใช้ (Client) ทําการเรียกรรมหน้าเว็บไซต์ได้โดยใช้โพรโทคอล HTTP ผ่านทางเว็บเบราว์เซอร์หรือ โปรแกรมคอมพิวเตอร์ ที่ทํางานตามคุณสมบัติข้างต้นการใช้งาน

1. เมื่อใช้นาแลต (URL) ในโปรแกรมเว็บเบราว์เซอร์ เช่น IE, Firefox, Google chome/ddh
2. เครื่องโดยแอนท์จะแปลงชื่อโฮสต์ ภายในยูนาและเป็นไอพีแอดเดรส
3. เครื่องใด แอนท์ติดต่อกับเครื่องเว็บเซิร์ฟเวอร์ โดยปรกติจะใช้โทรโทคอลTCP wada 89
4. เมื่อฟากเซียนต่อเสร็จ จะใช้โพรโทคอล HTTP ในการเรียกใช้ข้อมูลที่ต้องการ

    ซอฟต์แวร์เว็บเซิร์ฟเวอร์ที่ได้รับความนิยม คือซอฟต์แวร์เว็บเซิร์ฟเวอร์ที่ได้รับความนิยม คือ

    Apache HTTP Server จาก Apache Software FoundationInternet Information Server (115) จากไมโครซอฟท์ Sun Java System Web Server จากซัน ไมโครซิสเต็มส์ Zeus Web Server in Zeus Technology

โคลเอนต์ หรือ เครื่องลูกข่าย (อังกฤษ : client)

     เป็นระบบหรือแปรงเคชั่นที่สามารถเชื่อมต่อเข้ากับระบบคอมพิวเตอร์อื่น ที่เรียกว่าเซิร์ฟเวอร์ได้ คําว่าไคลแอนด์เริ่มมีการใช้เรียกตั้งคอมพิวเตอร์ที่ไม่สามารถเรียกใช้งานโปรแกรมคอมพิวเตอร์ในตัวเอง
แต่สามารถใช้งานโปรแกรมนั้นผ่านทางระบบเครือข่าย

ประโยชน์ของเว็บไซต์

1. ช่วยในศักยภาพการแข่งขันในด้านธุรกิจ
2. ช่วยสร้างภาพลักษณ์ที่ดี ความทันสมัย ความ า ล ให้กับองค์กร บริษัท และรกิจต่าง ๆ
3. ช่วยเผยแพร่ข้อมูลข่าวสารและบริการต่าง ๆ ให้เป็นทีรู้จักอย่างแพร่หลาย และช่วยลดค่าใช้จ่ายในการโฆษณาประชาสัมพันธ์
4. ช่วยทําหน้าที่เป็นที่ปรึกษา อํานวยความสะดวก ให้คําแนะนําต่าง ๆเกี่ยวกับสินค้าและบริการแก่ลูกค้าได้ตลอด 24 ชั่วโมง โดยไม่ต้องรอร้านเปิดให้บริการ
5. เป็นการเพิ่มช่องทางในการขายสินค้าและบริการของบริษัทอีกช่องทางหนึ่งช่วยให้เพิ่มยอดขายให้กับธุรกิจได้อย่างมีประสิทธิภาพ เข้าถึงกลุ่มลูกค้าได้จากทุกทีทั่วโan
6. เป็นการยกระดับมาตรฐานการซื้อขาย ทั้งภายในประเทศ และต่างประเทศ

ส่วนประกอบสําคัญของเว็บไซต์

ส่วนหัว (Header)

    สําหรับไว้แสดงโลโก้ หรือข้อความที่บ่งบอกว่า เว็บไซต์ของเรามีเนื้อหาอะไรบ้างในส่วนหัวอาจเป็นพื้นที่สําหรับใส่ป้ายโฆษณา สําหรับการหารายได้พิเศษนอกจากนี้อาจมีคํานิยามของเว็บไซด์
หรือสโลแกนต่อท้ายโลโก้ (อาจมีหรือไม่มีก็ได้) ถ้ามีอาจทําให้เกิดความคุ้นเคยจาง่าย น

เมนู (Navigator)}

     การมีเมนู ช่วยให้การค้นหาข้อมูลได้สะดวกมากขึ้น การวางเมนู ส่วนใหญ่จะอยู่ด้านบนด้านซ้าย หรือด้านขวา เป็นหลัก ส่วนหลายๆ เว็บในปัจจุบัน มีการเพิ่มเมนูในส่วนท้าย
อาจเป็นเพราะเนื้อหามีความยาวเกินหนึ่งหน้าหรือเพื่อเป็นการเพิ่มความสะดวกในการเข้าถึงเนื้อหาในส่วนอื่น ๆ อีกด้วย

เนื้อหา (Contents)

     หมายถึง ข้อมูลที่เราต้องการให้ผู้เยี่ยมชมรับทราบ เนื้อหาที่ดีควรมีการจัดแบ่งเป็นหมวดหมู่ให้ชัดเจน
ประกอบด้วย ข้อความที่เป็นตัวอักษร รูปภาพ / ภาพเคลื่อนไหว ไฟล์เสียง วีดีโอ ฯลฯ

ส่วนล่าง หรือส่วนท้าย (Footer)

     สําหรับแสดงรายละเอียดพื้นฐาน เช่น ใส่คําพูด ส่วนพืชสิทธิ์ (copyright) หรือใส่ที่อยู่ในการติดต่อ

ช่องว่าง (Space)

     เนื้อหาประกอบการจัดวางเว็ปไซต์ จะมีมาตรฐานดังตัวอย่างหากเรามีการปรับเปลี่ยนหรือย้ายตำแหน่ง อาจทำให้ผู้เยี่ยมชมเว็บไซต์ของเราเกิดความสับสนได้
ดังนั้นเพื่อเป็นการเริ่มต้นการทำเว็บที่ดี แนะนำาให้ทำตามดังตัวอย่างมาตรฐานก่อน

ส่วนประกอบย่อยอื่น ๆ ที่จำเป็น

1. Text เป็นข้อความปกติ โดยเราสามารถตกแต่งให้สวยงามและมีลูกเล่นต่าง ๆ
2. Graphic ประกอบด้วยรูปภาพ ลายเส้น ลายพื้น ล่าง
3. Multimedia ประกอบด้วยรูปภาพ ภาพเคลื่อนไหว และแฟ้มเสียง
4. Counter ใช้นับจํานวนผู้ที่เข้ามาเยี่ยมชมเว็บเพจของเรา
5. Cool Links ใช่เชื่อมโยงไปยังเว็บเพจของตนเองหรือเว็บเพจของคนอื่น
6, Fores เป็นแบบฟอร์มที่ให้ผู้เข้าเยี่ยมชม กรอกรายละเอียด แล้วส่งกลับมายังเว็บเพจ
7. Frames เป็นการแบ่งจอภาพเป็นส่วน ๆ แต่ละส่วนก็จะแสดงข้อมูลที่แตกต่างกันและเป็นอิสระจากกัน
8. Image Maps เป็นรูปภาพขนาดใหญ่ที่กําหนดส่วนต่าง ๆ บนรูป เพื่อเชื่อมโยงไปยังเว็บเพจอื่น ๆ
9. Java Applets เป็นโปรแกรมสําเร็จรูปเล็ก ๆ ที่ไม่ลงในเว็บเพจ เพื่อให้การใช้งานเว็บเพจมีประสิทธิภาพมากยิ่ง
